# EmpregAI

Encontre as melhores vagas em segundos.

O EmpregAI utiliza Web Scraping, Inteligência Artificial e análise ATS para encontrar oportunidades compatíveis com seu perfil, calcular sua aderência e gerar automaticamente um currículo personalizado para cada vaga.
## Tecnologias

- Python
- SQL Server
- SQLAlchemy
- Playwright
- Pandas

## Funcionalidades

- Busca automática de vagas
- Compatibilidade ATS
- Dashboard
- Currículo personalizado
- Cartas de apresentação automáticas
